# Archived Versions

This directory contains **frozen snapshots** of previous versions of the Tiny MCP Server.

## ⚠️ READ FIRST

- **DO NOT MODIFY** files in this directory
- **DO NOT USE** code from these versions in the current version (`/src`)
- These are **archived for reference only** to track architectural evolution
- The **current version is always in `/src`**

## Versions

### v1 - Initial Release
- SQLite database with hotel reviews
- Two tools: `query` and `listTables`
- Read-only database operations
- Claude Desktop integration via stdio
- See `/src` for current development
